建立網站程序

(1) 在 github 中，建立一個新的倉儲

(2) 在隨身系統中建立一個新的資料夾 tmp2

(3) 使用指令 git clone 下剛建立的倉儲

(4) 使用 git submodule add 裝入cmsimde 模組

(5) 進入 up_dir 將裡面的檔案複製到外面

(6) 進入cmsimde目錄下,將wsgi.py檔拉入SciTE中 執行

(7) 進入 https://127.0.0.1:9443/ 即完成建立網站
